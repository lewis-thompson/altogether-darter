This workspace is a React + Vite + TypeScript app that will use Firebase and Firestore.

Guidelines:
- Prefer incremental, file-specific changes rather than large rewrites.
- Use existing project structure and dependencies whenever possible.
- Keep React components simple, idiomatic, and easy to understand.
- When adding Firebase or Firestore code, follow current Firebase v10+ modular SDK patterns.
- Ask for clarification before implementing large features or architectural shifts.
- When making changes, list the modified files and include a concise summary.
- Avoid adding unrelated libraries unless explicitly requested.
